Nombre del elemento Descripcion uso
div El elemento HTML "div" es el contenedor genérico para el contenido de flujo. No tiene ningún efecto sobre el contenido o el diseño hasta que se le aplica estilo de alguna manera usando CSS (por ejemplo, se le aplica estilo directamente o se aplica algún tipo de modelo de diseño como Flexbox a su elemento principal). El elemento div debe usarse solo cuando ningún otro elemento semántico (como article o nav) sea apropiado.
"h1" Los elementos de encabezado implementan seis niveles de encabezado del documento, 'h1' es el más importante, y 'h6', el menos importante. Un elemento de encabezado describe brevemente el tema de la sección que presenta. La información de encabezado puede ser usada por los agentes usuarios, por ejemplo, para construir una tabla de contenidos para un documento automáticamente.
span Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea. Debe usarse solo cuando ningún otro elemento semántico sea apropiado. 'span' es muy parecido a un elemento 'div', pero 'div' es un elemento a nivel de bloque mientras que 'span' es un elemento en línea.
